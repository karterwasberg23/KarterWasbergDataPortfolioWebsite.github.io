Housing Prices and Average Salary Past 20 years (United States)

Created dynamic dashboards for data visualization in order to compare housing prices and occupation salaries.

I would like to analyze housing prices in the United States over the past 20 years to assess the affordability of homes today compared to previous years. Additionally, I am interested in examining the average salaries for various job types across different industries to better understand income trends in relation to housing costs.

  • Housing Price Data: Sourced from Zillow's Research Data. A separate Excel file was used to categorize the number of bedrooms, ranging from 1 to 5+.
  • Occupatinal Housing Data: Obtained from the U.S. Bureau of Labor Statistics. Excel files were downloaded for each year in May, dating back to 1993; however, only data from 2000 to 2023 was utilized for analysis.
  1. Converted each CSV file into a table format in Excel.
  2. Removed the "United States" row to exclude extra aggregated data
  3. Used Microsoft Power Query to unpivot columns that had "price estimate date" as header to achieve two columns: "Date of Price Estimate" and "Price Estimate."
  4. Added a column specifying the number of bedrooms for each table.
  5. Combined all tables into a single dataset using Power Query.
  1. Removed extra comments from each Excel sheet for easier data processing
  2. Added a "Year" column to each sheet.
  3. Appended data from 1997-2002 separately, as these years had different column headers compared to 2003-2023.
  4. Standardized column headers to ensure appending process would work properly.
  5. Merged the datasets using Microsoft Power Query.
  6. Exported the final dataset to SQL for further data cleaning.

Joining Table Summary Process

After appending the occupational wage data, some entries are missing occ_code values. Additionally, there are multiple job titles with slightly different titles. I wanted to categorize all job titles for data visualization in Tableau. I collected the new job titles from BLS Occupational Structure to obtain standardized occ_codes. This allowed me to map similar job titles to their corresponding codes, ensuring accurate categorization for older and newer job titles.

  1. Extracted unique occ_titles from the cleaned dataset (table: #distinctOcctitles).
  2. Extracted unique occ_titles from the U.S. Bureau of Labor Statistics website (table: #leveled_occupations).
  3. Joined both tables based on the number of matching words in job titles.
  4. Filtered out exact title matches and store them in a separate table (table: #perfect_match_occ_title_codes).
  5. Manually reviewed and filtered out mismatched results.
  6. Assigned occ_codes manually in Excel for remaining unmatched titles.
  7. Used the newly assigned occ_codes to categorize job titles.
