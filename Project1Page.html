Project 1
Page

Python, SQL, Tableau

The goal of this project is to design and develop an interactive dashboard that enables users to easily compare the prices and performance of various graphics cards. The dashboard will allow users to analyze key metrics such as benchmark scores and price-to-performance ratios, providing an intuitive interface for decision-making. By integrating recent data and dynamic visualizations, the dashboard will empower users to identify the best value graphics card options based on their specific needs and budget, whether for gaming, professional use, or general computing tasks.

The project involved gathering publicly available graphics card product data from a leading product website, which included detailed specifications such as brand, current price, previous price, and customer reviews. This data was then combined with relevant benchmark performance data sourced from another provider to develop a generic summary of the performance for each graphics card.

  • Python was used to perform web scraping to extract product from the website. Libraries such as BeautifulSoup and Requests allowed for efficient data extraction.
  • SQL was utilized to clean, organize, and join the data to ensure that the data was structured appropriately for analysis and visualization.
  • Tableau was used to create an interactive and insightful dashboard for visualizing the relationship between GPU prices and their performance benchmarks. It allowed for dynamic filtering and comparisons helping to communicate key findings in a clear and user-friendly manner.
